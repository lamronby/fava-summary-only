Summary Only Mode

This Fava instance is running in summary-only mode. Transaction-level details and editing features are not available.

Available reports

  • Balance Sheet
  • Trial Balance
  • Commodities
  • Events
  • Statistics

What is restricted

  • Income Statement (per-period transaction breakdown)
  • Journal (full transaction list)
  • Query (ad-hoc BQL queries)
  • Documents (attached files)
  • Editor (source file editing)
  • Import (file import tool)
  • Options (configuration viewer)
  • Per-account pages (transaction drilldown)
